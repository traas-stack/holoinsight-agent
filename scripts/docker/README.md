# 介绍
用于构建 agent docker 镜像.  
该脚本可以在 Mac m1 或 Linux 上执行, 要求机器上装有 docker. Mac m1 特别慢, 建议在 Linux 上执行, 可以通过 rsync 将代码复制到 Linux 上再构建.

```bash
./scripts/docker/build.sh your_agent_image:1.0.0
```
